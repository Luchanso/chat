# Chat
Simple chat on NodeJS

Example app: https://luchanso-chat.herokuapp.com/

```sh
npm install mocha -g
npm install socket.io express mocha should
```

or
```sh
npm i mocha -g
npm i
```

testing
```sh
npm i --save-dev
npm test
```

running
```
npm start
```

Open http://localhost:8080 in your browser
